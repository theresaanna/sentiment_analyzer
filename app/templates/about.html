About Sentiment Analyzer

The YouTube Sentiment Analyzer is a powerful tool designed to analyze and understand the sentiment of comments on YouTube videos.

Features

  • Extract comments from any public YouTube video
  • Perform sentiment analysis on comments
  • Visualize sentiment distribution
  • Identify trends and patterns in viewer feedback

How It Works

  1. Enter a YouTube video URL
  2. The system extracts the video ID
  3. Comments are fetched using the YouTube Data API
  4. Sentiment analysis is performed on each comment
  5. Results are aggregated and visualized

Technology Stack

This application is built with:

  • Flask - Python web framework
  • YouTube Data API v3 - For fetching video data
  • Bootstrap 5 - For responsive UI
  • Python 3.9+ - Core programming language
